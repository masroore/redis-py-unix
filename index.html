redis-py-unix by masroore

Redis Python Client

Fork of http://github.com/andymccurdy/redis-py/ Added support for Unix domain sockets

Install

host should point to the redis unix sock path. port = 0 i.e.: client = redis.Redis(host='/var/tmp/redis_sock', port=0, db=9)
